GeLo SDK

Develop powerful iOS applications that can interaction with GeLo Beacons.

Download this project as a .zip file Download this project as a tar.gz file

Discover Beacons

With the SDK, you can create apps that discover GeLo beacons throughout a site.

Reveal Content

GeLo's content management system bridges the gap between the digital world and physical world. Put content in your user's hands.

Go Offline

Sometimes internet connectivity with mobile devices isn't the greatest. Not to worry, with the GeLo SDK caches content to show to your uses at a later time.

Get Started

Getting up and running with the GeLo SDK is simple:

  1. Download and unzip the SDK.
  2. Open your XCode project, and drag the GeLoSDK.framework folder to your frameworks group.
  3. Read the documentation.
  4. Explore the example application
